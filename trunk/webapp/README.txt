it's a TPM demo web application deploying in one of web container like jetty. 
After deploying, you can use this URL to access the TPM enableing and not enabling login page. 

TPM enabling:
http://localhost:8082/tpm/identityservice/authn/login.jsp?&useTPM=true

TPM not enabling
http://localhost:8082/tpm/identityservice/authn/login.jsp

NOTICE:The port number may be vary according to different enviornment.
